# Mouse Gestures

A lightweight yet powerful browser extension that enhances your browsing efficiency through **intuitive mouse gestures**, **drag-and-drop actions**, and **cloud-synchronized settings**.

## ‚ú® Core Features

- **Mouse Gestures**: Hold the right mouse button and move the cursor to perform common actions such as forward, back,
  and closing tabs.
- **Drag Gestures**: Quickly search, open, or perform custom actions on selected text, links, and images.
- **Fully Customizable**: Create and manage custom gesture patterns and their associated actions.
- **Config Sync**: Backup and restore configuration via Google Drive for multi-device usage.
- **Lightweight and Efficient**: No ads, no unnecessary features, smooth interface, and low resource usage.

---

## üîí Privacy & Permissions

- The extension **does not collect or upload any browsing data**
- All gesture configurations and rules are stored locally on the user‚Äôs device
- Cloud sync is optional and stores configuration files in Google Drive only when enabled by the user
- All sensitive permissions are designed as **optional permissions**, requested only when related features are used
- No analytics, tracking, or third-party monitoring code included
